Aprendendo Sobre Bootstrap 5

Container-Fluid

Este container ocupa toda a largura da tela em qualquer tamanho.

Container-SM

Este container tem largura total até o breakpoint "sm" (576px). A partir disso, ele terá uma largura fixa.

Container-MD

Este container tem largura total até o breakpoint "md" (768px). Ideal para tablets e acima.

Container-LG

Este container tem largura total até o breakpoint "lg" (992px). Ideal para desktops.

Container-XL

Este container tem largura total até o breakpoint "xl" (1200px). Ideal para telas grandes.

Breakpoints

Os breakpoints definem os pontos onde o layout muda com base no tamanho da tela. São ideais para criar design responsivo.

Containers

Containers controlam a largura máxima dos elementos. Use container ou container-fluid para definir o comportamento.

Grid

O sistema de grid do Bootstrap organiza o conteúdo em linhas e colunas, permitindo criar layouts complexos e responsivos.

Columns

As colunas são os blocos de construção do grid. Use classes como col-md-6, col-sm-4 para definir o tamanho em cada breakpoint.

Entendendo as Colunas do Bootstrap

As colunas são os blocos de construção fundamentais do sistema de grid do Bootstrap. Elas são sempre colocadas dentro de uma .row e usam classes como .col-* para definir o tamanho em diferentes breakpoints.

Comportamento Básico das Colunas

Se você não especificar uma largura para as colunas, elas se dividirão igualmente pelo espaço disponível.

Coluna 1
Coluna 2
Coluna 3

Neste exemplo, cada coluna ocupa 1/3 da largura, independentemente do tamanho da tela.

Colunas com Larguras Definidas

Você pode usar as classes .col-* para definir a largura das colunas em diferentes breakpoints. O asterisco representa um número de 1 a 12, que indica quantas colunas o elemento deve ocupar.

col-md-3 (25%)
col-md-6 (50%)
col-md-3 (25%)

Em telas médias (md) e maiores, a primeira e a terceira colunas ocuparão 25% da largura, enquanto a segunda coluna ocupará 50%.

Colunas Responsivas

Você pode combinar classes .col-* para criar layouts responsivos que se adaptam a diferentes tamanhos de tela.

col-sm-6 col-md-4
(50% em telas pequenas, 33.33% em telas médias e maiores)
col-sm-6 col-md-4
(50% em telas pequenas, 33.33% em telas médias e maiores)
col-sm-12 col-md-4
(100% em telas pequenas, 33.33% em telas médias e maiores)

Em telas pequenas (sm), cada coluna ocupará 50% da largura. Em telas médias (md) e maiores, cada coluna ocupará 33.33% da largura. A terceira coluna ocupará 100% da largura em telas pequenas.
